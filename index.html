سياسة الخصوصية لامتداد مساعد كلمات Redbubble

تاريخ السريان: ١ يناير ٢٠٢٤

١. مقدمة

يمثل امتداد "مساعد كلمات Redbubble" (يُشار إليه فيما يلي بـ "الامتداد") التزامنا الكامل بحماية خصوصية المستخدمين. لا نقوم بجمع أو مشاركة أي بيانات شخصية.

٢. البيانات التي نجمعها

الامتداد لا يجمع أي بيانات شخصية. البيانات التي يتم حفظها محليًا تشمل:

  • مشاريع الكلمات المفتاحية التي تحفظها (تخزن على جهازك فقط)
  • مفتاح Gemini API (إذا قمت بإدخاله، يخزن محليًا فقط)
  • إعدادات التفضيلات الخاصة بك

٣. التخزين والوصول إلى البيانات

يستخدم الامتداد نظام التخزين المحلي لمتصفح Chrome (chrome.storage) فقط. جميع البيانات تبقى على جهازك ولا يتم إرسالها إلينا أو إلى أي طرف ثالث.

٤. الاتصالات الخارجية

لأداء وظائفه، يتصل الامتداد بـ:

  • Google Suggest API - لجلب اقتراحات الكلمات المفتاحية
  • Google Gemini API - لتوليد المحتوى (باستخدام مفتاح API الخاص بك فقط)
  • موقع Redbubble - للتفاعل مع واجهة الموقع

لا يتم إرسال أي بيانات شخصية في هذه الطلبات.

٥. الأذونات والصلاحيات

يطلب الامتداد الصلاحيات التالية لأداء وظائفه:

  • activeTab: للوصول إلى صفحة Redbubble النشطة لعرض الاقتراحات وتعبئة النماذج
  • scripting: لحقن نصوص برمجية صغيرة في صفحات Redbubble للتفاعل مع واجهة الموقع
  • storage: لتخزين إعداداتك ومشاريعك محليًا على جهازك
  • host_permissions: للاتصال بـ Redbubble وGoogle APIs لتقديم الخدمة
